This folder for codes of detection of drowsy driving

The ipynb files start with "eye" and "yawn" are for model training.

The ipynb files start with "combine" are the results of test set.

eye_model.h5 is the model to detect close eyes.

yawn_3.h5 is the model to detect yawning.

haarcascade.xml, haarcascade_eye.xml, haarcascade_frontalface_default.xml, the three files are the code to achieve the face recognizer and eye recognition required files
